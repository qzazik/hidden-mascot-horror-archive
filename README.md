# Hidden Mascot Horror Archive

Локальный каталог малоизвестных mascot horror игр на React, Vite и TypeScript.

## Что уже есть

- Главная страница с подборками и быстрым входом в рулетку.
- Каталог с поиском, сортировкой, фильтрами и экспериментальной подборкой.
- Страницы игр, разработчиков, серий, списка пользователя и рулетки.
- Сохранение отметок, истории рулетки и решений импорта в `localStorage`.
- Скрытая панель `/admin/import` для ручной проверки Excel-импорта.
- Адаптивная верстка для ПК, планшета и телефона.

## Запуск

```bash
npm install
npm run organize:assets
npm run import:games
npm run dev
```

## Сборка и тесты

```bash
npm run build
npm test
```

## Импорт игр из Excel

Поместите `Hidden_Mascot_Horror_50_Games.xlsx` в корень проекта или в `data-import/`, затем выполните:

```bash
npm run import:games
```

Скрипт `scripts/import-games-from-excel.mjs`:

- читает все листы и ищет строку игровых заголовков;
- нормализует записи и ищет дубликаты, не объединяя Demo, Chapter, Classic, Remake и другие версии;
- сопоставляет таблицу с текущим каталогом и локальными изображениями;
- записывает старые оценки только в `legacyRatings`;
- не меняет публичную базу игр.

Отчёты записываются в `data-import/` и зеркалируются в `public/data-import/` для админ-панели. Откройте `/admin/import`, выберите решения и экспортируйте JSON. До ручного подтверждения новые игры не попадают в публичный каталог.

## Редакционные оценки

Скрытая страница `/admin/ratings` позволяет выбрать игру, заполнить критерии, вердикт и статус проверки. Общий рейтинг вычисляется автоматически только при наличии не менее пяти критериев.

Кнопка «Сохранить локально» записывает данные в `localStorage` под ключом `hmha:editorial-ratings:v1`. Это даёт локальный предпросмотр, но не меняет опубликованный сайт.

Для переноса оценок:

1. Нажмите «Экспортировать ratings.json».
2. Замените файл `src/data/ratings.json` экспортированным `ratings.json`.
3. Проверьте `npm run build` и `npm test`.
4. Выполните `git add`, `git commit` и `git push` в своём репозитории.

Личные оценки посетителя хранятся отдельно в `hmha:personal-ratings:v1` и никогда не участвуют в редакционном рейтинге.

## Локальные изображения

`scripts/organize-assets.mjs` раскладывает изображения по `public/assets/games/<slug>/cover`, `hero` и `gallery`. Нераспознанные файлы попадают в `public/assets/unsorted`.

## Публикация

Проект использует `BrowserRouter` и `public/404.html` для маршрутов GitHub Pages. Для публикации в подкаталоге задайте `VITE_BASE_PATH` перед сборкой.

## Примечания по данным

- Неподтверждённые поля показываются как `Не указано`, `Требует проверки` или `Не проверено`.
- Оценки Excel считаются низкодоверительными и не отображаются как официальный рейтинг сайта.
